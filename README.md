# PHP Login System

A secure login system built using PHP, Bootstrap 5, Sessions, Cookies, and User-Based Themes.

## 🚀 Features

- User Authentication
- PHP Sessions
- Remember Me Cookies
- User-Based Themes
- Bootstrap 5 Responsive UI
- Dashboard Protection
- Custom Validation Functions
- Multiple User Login Support

---

```bash
Login_System/

```
